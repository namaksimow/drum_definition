Положи сюда иконки для карточек сообщества:

- `comment.png`
- `like.png`
- `fire.png`
- `wow.png`

Рекомендуемые параметры:

- формат: `PNG` с прозрачным фоном (альтернатива: `SVG`)
- базовый размер файла: `32x32 px`
- отображаемый размер в интерфейсе: `16x16 px`
- вес одной иконки: до `20 KB`

Если используешь SVG:

- viewBox: `0 0 32 32`
- без встроенных шрифтов
- без внешних зависимостей (стили внутри файла)
